## Implementation
**Dynamic Programming** algorithms share a common structure based on four main principles:
1. Characterizing the optimal substructure of the problem.
2. Defining the value of an optimal solution recursively.
3. Computing the value of an optimal solution in a bottom-up or top-down fashion.
4. Constructing an optimal solution from the computed information.
